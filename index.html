luxweather

luxweather is an APACHE licensed Twitter weather bot written in Python. It is designed to tweet weather conditions and forecasts from Yahoo! Weather.

Installation:

From source use

$ python setup.py install

or install from PyPi

$ pip install luxweather

Edit the config.py file to suit your needs and create a cron job like this:

$ 30 * * * * python /path/to/luxweather.py
